Ce mini‑site web a été réalisé dans le cadre du TD4 de Développement Web (L2 Informatique – S4).
Il présente notre projet de Génie Logiciel, centré sur la simulation du mécanisme d’ADN, la génétique,
et la visualisation pédagogique des processus biologiques.
Le site a été conçu en HTML5, CSS3, et vérifié en XHTML pour garantir une structure strictement bien formée.

Équipe du projet : 
-Benkhaled Malia
-Iadaden Dalia
-Achour Marie

Le site contient 6 pages, chacune dédiée à une partie du projet :

-index.html : Page d’accueil
-membres.html : Présentation de l’équipe
-description.html : Description et périmètre du projet
-specifications.html : Spécifications fonctionnelles et techniques
-fonctionnalites.html : Fonctionnalités attendues
-contraintes.html : Contraintes légales, techniques, éthiques et pratiques

Chaque page contient un menu de navigation identique, ainsi qu’un logo cliquable permettant de revenir à l’accueil.

Technologies utilisées:
-HTML5 : structure du site
-CSS3 : mise en forme (feuille de style externe style.css)
-XHTML5 (validation XML) : vérification de la bonne formation des pages
-Balises HTML5 modernes : header, main, nav, section, article, figure, figcaption, footer, etc.
